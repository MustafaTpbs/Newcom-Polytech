L’observatoire de Nançay a développé un banc de test pour les cartes dites NEWCOM afin de valider leur intégration dans les systèmes finaux. Il y a une soixantaine de cartes qui seront intégrées dans le Radiohéliographe de l’Observatoire Radioastronomique. 
	Le banc de test actuel fonctionne en envoyant des commandes à un générateur de signal qui est relié à un PCB d’interfaçage. L’opérateur branche la carte NEWCOM sur ce PCB d’interfaçage afin d’en observer les sorties sur un logiciel graphique installé sur son PC.
	La problématique : l’opérateur doit systématiquement manipuler un système de câbles volants afin de régler le banc de test sur ce qu’il veut observer. De plus, l’observation se fait sur un logiciel graphique qui affiche les signaux de sortie, donc l’opérateur doit obligatoirement avoir les compétences pour les analyser de lui-même. Cela est également très chronophage, car il doit le faire pour chaque signal, pour chaque carte.
